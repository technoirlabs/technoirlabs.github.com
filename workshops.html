Tech for non-tech founders

This intensive one day workshop is intended for non-tech company founders or project managers who need a rapid overview of contemporary web development.

Material covered includes

  • Domain registration, email and social media account setup
  • Choosing a technology stack
  • Mobile & app development
  • Web analytics and user testing
  • Managing an effective development project

No prior background in tech or web development is assumed, however it is expected that you are comfortable with browsing the web and sending/receiving email.
